入手小米平板一代

没错，现在是 2018 年，我入手了一款 2015 年的电子产品。

这周五的时候购买了小米平板一代。为了这个平板，我又买了个罗技的 k480 蓝牙无线键 盘，今天（周末）刚刚拿到。这两个产品都是在咸鱼上购买的，且都是面交，具体情况如 下：

  • 小米平板： 360 元，包含原装充电器，保护套，贴膜。
  • k480 键盘： 100 元，包含两节七号电池，键盘包。

最近天冷了，早上经常把笔记本放在小折叠桌上，然后躺在床上玩。之所以买个平板，主 要的原因是早上躺在床上基本都是用笔记本看看文档，学学英文，需求比较轻量，另一方 面是笔记本上有一块机械硬盘，而在床上玩电脑容易对转动中的机械硬盘造成损害。所以 想换个方案来满足我的需求，于是自然就想到了平板。

关于平板的选择，我考虑最重要的一点是能否安装 Linux 系统。因此在 Windows 平板和 Android 平板的选择之间纠结了一阵。理论上 Windows 平板（类似于 Window是电脑）可 以安装任何 Linux 系统，但实际只有少部分机型可以直接安装 Linux 系统，部分机型即 使装上 Linux 系统，也可能出现各种不兼容问题。 Android 平板虽不能直接安装 Linux 系统，但是有相应的 APP 可以模拟 Linux 系统（Termux, Linux Deploy 等）。之后在了 解 Windows 和 Android 相关机型之后，发现可以安装 Linux 系统，且兼容性比较好的 Windows 平板价格还是比较高的，其实这样的平板更像是一款 Windows 轻薄本。

最终我还是偏向选择 Android 平板，主要原因有两点：第一点正如之前提到的价格因素， 第二点是我一直想学习一下 Andorid 如何刷机（虽然早在多年前刷过一台 HTC 手机，但 也仅此一例，缺乏更多的实战经验）。

要论刷机，网上讨论最多的应该还属小米，后来我又好奇用 google 搜了一下 xiaomi flash rom 这几个关键字，没想到小米在国外也是相当受欢迎。这坚定了我买小米的想法 。（后来我才发现不论在国内还是在国外，小米都吸引了一帮子手机发烧友的追捧，我想 这主要的原因是，相对于其他手机，小米更加自由，同时官方提供了专业的刷机软件，教 程，对喜欢折腾手机的用户提供了极大的便利。)
