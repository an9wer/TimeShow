= getent

----
    getent [option]... database [key]...
----

== Examples

List all hosts:

----
    $ getent hosts
----


List all users:

----
    $ getent passwd
----


List all groups:

----
    $ getent group
----


List all services (may not be installed):

----
    $ getent services
----
