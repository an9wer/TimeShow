= Bash Tricks

-   Redirect output to log file and standard output:

----
    exec > >(tee -ai ${ME_JOB_RCONLE_LOG})
    exec 2>&1
----

thx: https://stackoverflow.com/a/3403786

-   Compare two directory trees

----
    diff -qr dir1/ dir2/
----
