Tabpage

See :h tabpage

Commands

{count}gt | :{count}tabn[ext]

Go to tab page {count}. The first tab page has number one.

{count}gT | :{count}tabp[revious]

Go to the previous tab page.

CTRL-W gf

Open a new tab page and edit the file name under the cursor.
