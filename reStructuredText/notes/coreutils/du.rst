Coreutils du
============

Scan only for specified depth: ::

    # du -d <depth> <file> [...]

Print only a total size of files (same as ``-d 0``): ::

    # du -s <file> [...]

List all files, not just directories (by default): ::

    # du -a <file> [...]

Do not count size of subdirectories into directories: ::

    # du -S <file> [...]
