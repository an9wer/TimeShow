WebDAV
======

Definition of WebDAV:

-   https://en.wikipedia.org/wiki/WebDAV

Deploy WebDAV on CentOS 7:

Install httpd and creat realated directory:

::

    # yum install httpd
    # mkdir /var/www/webdav
    # chown -R apache:apache /var/www/webdav
